# 프로젝트 Dash 핵심 가치 제안서 (Core Value Proposition)
**버전**: v1.1.0 (UI 고도화 반영)
**작성일**: 2026-03-12
**문서 목적**: 전국 아동보호전문기관 상담원을 위한 국가아동학대정보시스템 DB 자동 입력 솔루션 'Dash'의 As-Is vs To-Be 변화상 및 3대 기능적 핵심 가치 정의

---

## 1. Executive Summary: "복잡한 DB 입력을 Dash 하세요"
전국 아동보호전문기관 상담원들은 아동의 생명과 안전을 지키는 본연의 업무만큼이나, **'조사 및 상담 내용을 국가시스템에 수동으로 옮겨 적는 단순 반복 작업'**에 엄청난 시간과 에너지를 쏟고 있습니다. 

프로젝트 **Dash**는 엑셀 메모와 브라우저 확장 기술(RPA)을 융합하여, 매일 1~2시간씩 낭비되던 기계적 타이핑과 클릭 노동을 **단 5분 컷**으로 압축합니다. 특히 최신 업데이트를 통해 기존의 번거로운 드롭다운 인터페이스를 **'원클릭 버튼(Chip)'** 방식으로 전면 개편하여 입력 속도를 극대화했습니다.

---

## 2. As-Is vs To-Be: 현장의 패러다임 전환

| 구분 | As-Is (현재의 고통) | To-Be (Dash 도입 후의 해방) |
| :--- | :--- | :--- |
| **작업 방식** | 듀얼 모니터에 엑셀과 시스템 창을 띄워 놓고 한 땀 한 땀 **복사(Ctrl+C) & 붙여넣기(Ctrl+V)**를 반복함 | 엑셀 업로드 또는 전용 폼 입력 후 **'전체 창에 DB 입력' 버튼 클릭 한 번**으로 모든 윈도우 일괄 처리 |
| **UI 탐색** | 각 항목마다 드롭다운 메뉴를 클릭하고 리스트에서 원하는 항목을 찾아 다시 클릭해야 함 (최소 2-3회 클릭) | 모든 옵션이 **그리드 버튼(Chip) 형태로 노출**되어 있어, 눈으로 보고 즉시 **'원클릭'**으로 선택 완료 |
| **다중 창 관리** | 여러 명의 아동을 입력할 때 브라우저 탭을 일일이 찾아다니며 데이터가 섞이지 않을까 불안해함 | **'다중 창 캐러셀(Carousel)'** 인터페이스를 통해 여러 대상자의 창을 한 화면에서 스와이프하며 관리 |
| **소요 시간** | 10건 입력 기준 수천 번의 클릭과 오타 확인 발생. **평균 1~2시간 소요 (야근 유발)** | 고밀도 UI와 자동 기입 로직으로 **10건 입력에 5분 내외**로 완료. (클릭 횟수 80% 감소) |
| **사용자 감정**| 무력감, 퇴근 지연에 대한 스트레스, 번아웃 | 업무 효능감 상승, 본질적 상담 업무에 대한 집중도 향상 및 칼퇴근 보장 |

## 3. 3대 기능적 핵심 가치 (Core Functional Values)

### Value 1. 다중 창 병렬 처리 (Parallel Multi-Window Processing)
Dash는 사용자 환경에 열려 있는 여러 개의 시스템 창을 지능적으로 인식합니다.
- **캐러셀 뷰어**: 여러 아동의 정보를 입력할 때 탭을 옮겨 다닐 필요 없이, Dash의 캐러셀 UI를 넘기기만 하면 연결된 실제 시스템 창이 자동으로 동기화됩니다.
- **전체 창 일괄 입력**: 수동 입력이나 엑셀을 통해 준비된 데이터를 **'전체 창에 DB 입력'** 버튼 하나로 모든 활성화된 윈도우에 동시에 기입합니다. 1명을 넣는 시간으로 10명을 처리할 수 있습니다.

### Value 2. 인지적 마찰을 없앤 원클릭 버튼 UI (Instant-Click Interface)
기존 드롭다운 방식의 '리스트 탐색-선택' 과정을 제거하고, 모든 옵션을 화면에 직관적으로 배치했습니다.
- **고밀도 칩(Chip) 그리드**: 제공방법, 장소 등 자주 사용하는 항목을 2x2 또는 3x1 그리드 버튼으로 배치하여, 사용자는 메뉴를 열지 않고도 원하는 항목을 즉시 클릭할 수 있습니다.
- **스티키 액션 바**: 입력 폼이 길어져도 'DB 입력' 버튼은 항상 하단에 고정되어 있어, 스크롤 없이 언제든 입력을 실행할 수 있습니다.

### Value 3. 휴먼 에러 원천 차단 및 시각적 피드백 (Error-Free feedback)
국가 데이터베이스의 정확성을 담보하기 위해 기계 수준의 정밀도를 제공합니다.
- **비독점적 시각적 검증**: 데이터가 기입된 필드는 즉시 연두색 테두리로 강조되어 기입 성공 여부를 실시간으로 알립니다.
- **자동 포커싱**: 특정 창을 선택하거나 캐러셀을 넘기면, 해당 창 내부의 실제 입력 영역으로 자동 스크롤/포커싱되어 사용자가 "내가 지금 어디에 입력하고 있는가"를 놓치지 않게 합니다.

## 4. 제안 결론 (Conclusion)
**"Dash는 상담원의 시간을 데이터 노역이 아닌, '아이들의 미래'를 설계하는 데 쓰게 합니다."**

Dash는 단순히 업무를 약간 편하게 해주는 툴이 아닙니다. **월 5만 원 미만**의 저렴한 비용으로 **기관 소속 상담원 전체의 업무 효율을 수치상 1200% 이상 향상**시키는 혁신적인 업무 자동화 인프라입니다. 이제 드롭다운을 뒤적이는 대신, Dash 버튼 하나로 업무를 끝내고 아이들의 목소리에 집중하십시오.
