## 📊 아동학대정보시스템 사용 기관 전체 현황

### 1️⃣ **아동보호전문기관** — 102개소

**역할**: 아동학대 신고 접수, 현장 조사, 사례 판단, 서비스 제공

| 항목 | 현황 |
|---|---|
| 전국 기관 수 | 102개소 |
| 법인 분포 | 굿네이버스(46), 세이브더칠드런(8), 초록우산(7), 기타 |
| 시스템 사용 업무 | 신고 접수, 조사 기록, 사례 관리, 서비스 제공 내역 |
| **Linker 타겟** | ✅ **핵심 1차 타겟** (현재 대전 기관에서 근무) |

**비즈니스 임팩트**:
- 기관당 연 60만 원 × 102개 = **6,120만 원**
- 굿네이버스 본부 계약 시 46개 일괄 도입 가능

### 2️⃣ **지자체 아동학대전담공무원** — 229개 시군구 / 885명

> **2020년 10월 아동학대 조사 공공화** — 민간 아보전에서 지자체로 조사 권한 이관

**역할**: 아동학대 현장 조사, 학대 여부 판단, 피해 아동 보호 조치

| 항목 | 현황 |
|---|---|
| 전국 시군구 | 229개 (전국 252개 시군구 중 91%) |
| 전담공무원 수 | 885명 (2024.12 기준) |
| 시스템 사용 업무 | 현장 조사 기록, 학대 판단 결과 입력, 보호 조치 기록 |
| 협력 기관 | 경찰과 공동 현장 출동, 아보전과 정보 공유 |
| **Linker 타겟** | ✅ **2차 타겟** (B2G 영업) |

**시스템 접근 권한**:
- 지자체 공무원은 **아동학대정보시스템**에 직접 접근
- 경찰 및 아보전과 실시간 정보 공유
- 조사 결과, 보호 조치, 사후 관리 내역 모두 시스템 입력

**비즈니스 임팩트**:
- **시군구 단위 계약**: 229개 × 60만 원 = **13,740만 원**
- **개인 라이선스 판매**: 885명 × 30만 원 = **26,550만 원**
- 공공 입찰 or 수의계약 방식으로 접근

### 3️⃣ **경찰서** — 전국 259개소

**역할**: 아동학대 수사, 현장 동행 조사, 경찰청 APO 시스템과 연동

| 항목 | 현황 |
|---|---|
| 전국 경찰서 수 | 259개소 (2023.08 기준) |
| 시스템 연동 | **경찰청 APO 시스템 ↔ 국가아동학대정보시스템** |
| 시스템 사용 업무 | 112 신고 접수 정보 공유, 현장 조사 기록 공유 |
| 협력 방식 | 지자체 전담공무원과 공동 현장 출동 |
| **Linker 타겟** | △ **3차 타겟** (협력 기관, 직접 판매는 어려움) |

**시스템 접근 권한**:
- 경찰청 자체 시스템(APO)과 아동학대정보시스템이 **통합 연동**
- 112 신고 정보를 아동학대정보시스템으로 자동 전송
- 지자체/아보전과 실시간 정보 공유

**비즈니스 임팩트**:
- 경찰은 자체 시스템 사용 → **직접 판매 어려움**
- 그러나 협력 기관으로서 **우리 프로그램의 가치를 증명**하는 레퍼런스로 활용 가능
- "경찰-지자체 협력 강화"를 위한 데이터 표준화 솔루션으로 포지셔닝 가능

## 🚨 **가장 중요한 발견**

### 2020년 아동학대 조사 공공화의 의미

2020년 10월 이전:
- 아동학대 조사 = **민간 아보전 담당**
- 시스템 사용자 = 아보전만

2020년 10월 이후:
- 아동학대 조사 = **지자체 공무원 담당**
- 시스템 사용자 = **아보전 + 지자체 + 경찰 + 병원**

**이것이 의미하는 것**:
- 아동학대정보시스템은 이제 **국가 공공 인프라**
- 단순 민간 아보전 도구 → **범정부 통합 플랫폼**으로 격상
- **공공 입찰 시장** 진입 가능
- **보건복지부 주도 전국 확산** 가능성
