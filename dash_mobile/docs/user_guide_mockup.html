DASH 한눈에 보기

현장 기록부터 시스템 입력까지
5단계로 완성됩니다.

STEP 1
새 사례 등록
홈 화면 하단의 + 버튼을 눌러 새 사례를 만드세요.
아동 이름과 동(읍·면·동) 정보를 입력하면 사례가 생성됩니다.
💡 아동 이름은 마스킹되어 외부에 노출되지 않습니다.
STEP 2
상담 기록 작성
등록된 사례를 탭하면 상담 기록 폼이 열립니다.
제공 구분, 방법, 서비스 유형, 일시, 내용 등을 입력하세요.
태블릿과 휴대폰 모두 지원합니다.
💡 저장한 기록은 서버 연결 없이도 기기에 보관되며, 연결 복구 시 자동 동기화됩니다.
STEP 3
상사에게 공유 및 검토 요청
작성 완료 후 공유 버튼을 누르면 고유 링크가 생성됩니다.
링크를 상사에게 전달하면 PC 웹에서 내용을 확인하고
수정·검토를 진행할 수 있습니다.
💡 링크는 본인 계정의 기록에만 접근 가능합니다.
STEP 4
검토 완료 알림 수신
상사가 검토를 완료하면 즉시 푸시 알림이 도착합니다.
홈 화면 알림 탭에서 검토 내용을 확인할 수 있습니다.
💡 알림 설정은 프로필 탭에서 변경할 수 있습니다.
STEP 5
NCADS 자동 입력
PC에서 Chrome 확장프로그램을 설치하세요.
NCADS 업무 시스템에 접속한 상태에서 확장프로그램을 열면
작성한 기록이 클릭 한 번으로 자동 입력됩니다.
💡 Chrome 웹 스토어에서 무료로 설치할 수 있습니다.
