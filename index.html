Dash — 개인정보처리방침

최종 수정일: 2026년 2월 26일

1. 수집하는 개인정보

Dash는 어떠한 개인정보도 수집하지 않습니다.

  • 사용자가 업로드하는 엑셀 파일은 브라우저 내부(로컬)에서만 처리되며, 외부 서버로 전송되지 않습니다.
  • 확장프로그램 사용 기록은 Chrome 로컬 스토리지에만 저장되며, 개발자에게 전달되지 않습니다.

2. 데이터 처리 방식

  • 엑셀 파일 파싱은 브라우저 내 JavaScript(SheetJS 라이브러리)로 수행됩니다.
  • 파싱된 데이터는 대상 시스템의 입력 폼에 자동 기입하는 용도로만 사용됩니다.
  • 모든 데이터는 브라우저 세션 종료 또는 사용자의 명시적 삭제 시 제거됩니다.

3. 외부 통신

본 확장프로그램은 어떠한 외부 서버와도 통신하지 않습니다. 네트워크 요청이 발생하지 않습니다.

4. 권한 사용 목적

  • tabs: 대상 시스템 창을 감지하기 위해 사용합니다.
  • activeTab: 현재 활성 탭에 데이터를 자동 입력하기 위해 사용합니다.
  • scripting: 대상 페이지의 입력 폼에 값을 채우기 위해 사용합니다.
  • storage: 사용자의 입력 기록을 로컬에 저장하기 위해 사용합니다.
  • sidePanel: 사이드 패널 UI를 제공하기 위해 사용합니다.
