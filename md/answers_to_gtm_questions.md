## 🟢 기술 및 제품 관련

### Q4. 국가아동학대정보시스템에 접속할 때 브라우저는?

**답변**: **Microsoft Edge 대부분 사용**

**임팩트**:
- Edge = Chromium 기반 → **Selenium / Playwright 안정적 지원**
- 크롬과 거의 동일한 DOM 구조 → 개발 난이도 낮음
- Edge WebDriver 공식 지원 → 유지보수 용이

**기술 스택 확정**:
```python
# Edge WebDriver 사용
from selenium import webdriver
from selenium.webdriver.edge.service import Service
from selenium.webdriver.edge.options import Options

edge_options = Options()
edge_options.add_argument("--start-maximized")
driver = webdriver.Edge(service=Service(), options=edge_options)
```

### Q7. 팝업창에서 Ctrl+V 붙여넣기가 정상 작동하나요?

**답변**: ✅ **정상 작동**

**임팩트**: 🎉 **가장 중요한 기술 검증 통과!**

- Ctrl+V 작동 = **클립보드 방식 자동화 가능**
- 보안 프로그램이 클립보드 차단하지 않음
- 개발 난이도 대폭 감소

**기술 구현 방식**:
```python
import pyperclip
from selenium.webdriver.common.keys import Keys

# 데이터를 클립보드에 복사
pyperclip.copy("입력할 텍스트")

# 필드를 클릭하고 Ctrl+V로 붙여넣기
field = driver.find_element(By.ID, "field_id")
field.click()
field.send_keys(Keys.CONTROL, 'v')
```

**RPA 로직 확정**:
1. 엑셀에서 데이터 읽기
2. 클립보드에 복사
3. 시스템 필드 클릭
4. Ctrl+V로 붙여넣기
5. 다음 필드로 이동

### Q10. 서비스 이름 "Linker"에 확정인가요?

**답변**: ❌ **Linker는 확정 아님**
- 현재 프로젝트명: **dbauto**
- 아직 최종 이름 미확정

**브랜딩 옵션**:

| 후보 이름 | 의미 | 장점 | 단점 |
|---|---|---|---|
| **Linker** | Link(연결) + er | 간결함, 글로벌 느낌 | 너무 추상적 |
| **자동기입** | 그대로 자동 기입 | 직관적, 기능 명확 | 촌스럽게 느껴질 수 있음 |
| **케이스링크** | Case + Link | 사회복지 전문성 | 발음 어려움 |
| **DB메이트** | DB + Mate | 친근함, 도우미 느낌 | DB가 기술 용어라 거부감 |
| **아이다** | "아동 입력 다 해줍니다" | 한국적, 친근함 | 너무 캐주얼 |
| **기록비서** | 기록 + 비서 | 명확한 역할 정의 | 평범함 |
| **온기록** | 온라인 + 기록 | 부드러움, 기억하기 쉬움 | 다른 서비스와 혼동 가능 |

**추천**: 
1. **1순위: 케이스메이트 (CaseMate)** - "사례 관리 친구"
2. **2순위: 온기록 (OnRecord)** - "온라인으로 기록을"
3. **3순위: 자동기입 (Auto-Entry)** - 직관적이지만 브랜드 없음

**네이밍 결정 프로세스**:
