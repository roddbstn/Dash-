# 프로젝트 Linker 핵심 가치 제안서 (Core Value Proposition)
**버전**: v1.0.0
**작성일**: 2026-02-21
**문서 목적**: 전국 아동보호전문기관 상담원을 위한 국가아동학대정보시스템 DB 자동 입력 솔루션 'Linker'의 As-Is vs To-Be 변화상 및 3대 기능적 핵심 가치 정의 (내부 보고 및 제안서 작성용)

---

## 1. Executive Summary: 우리가 바꾸려는 300분의 삶
전국 아동보호전문기관 상담원들은 아동의 생명과 안전을 지키는 본연의 업무만큼이나, **'조사 및 상담 내용을 국가시스템에 수동으로 옮겨 적는 단순 반복 작업'**에 엄청난 시간과 에너지를 쏟고 있습니다. 
프로젝트 Linker는 엑셀 메모와 브라우저 확장 기술(RPA)을 융합하여, 매일 1~2시간씩 낭비되던 기계적 타이핑과 클릭 노동을 **단 5분 컷**으로 압축합니다. 우리는 상담원의 손가락을 노동에서 해방시켜, 그들이 다시 아동들의 눈을 맞출 수 있게 합니다.

## 2. As-Is vs To-Be: 현장의 패러다임 전환

| 구분 | As-Is (현재의 고통) | To-Be (Linker 도입 후의 해방) |
| :--- | :--- | :--- |
| **작업 방식** | 듀얼 모니터에 엑셀과 시스템 창을 띄워 놓고 한 땀 한 땀 **복사(Ctrl+C) & 붙여넣기(Ctrl+V)**를 반복함 | 엑셀 기반 데이터(또는 수동 패키지 폼)를 로드하여 **클릭(Click) 한 번에 일괄 전송(Auto-Fill)**함 |
| **소요 시간** | 10건 입력 기준 분당 약 30회의 클릭 및 드롭다운 탐색 발생. **평균 1~2시간 소요 (야근 유발)** | 다중 창 병렬 처리 및 Hover-to-Open 동선 최적화로 **10건 입력에 5~10분 이내 무결점 완료** |
| **다중 선택** | 담당자/대상자가 3명일 경우 드롭다운을 9번 클릭해야 하며, 이 과정에서 시스템 지연으로 추가가 누락되기도 함 | **Click-Zero 리스트 빌더** 적용. 마우스 오버만으로 메뉴가 열리고 클릭 한 번에 명단이 확정됨 |
| **정확도** | 단순 반복 노동 과정에서 피로 누적으로 인한 휴먼 에러(동명이인 오기입, 제공일시/장소 선택 누락) 발생 | 정밀한 DOM 타겟팅과 비동기 순차 처리(150ms 딜레이)로 기계 단위의 **오류율 0%** 보장 |
| **사용자 감정**| 무력감, 퇴근 지연에 대한 스트레스, 번아웃 | 퇴근 시간 보장에 따른 심리적 안정, 본질적 상담 업무에 대한 집중도 향상 |

## 3. 3대 기능적 핵심 가치 (Core Functional Values)

### Value 1. 극단적 입력 시간 단축 (Hyper-Efficiency)
단순한 '매크로'가 아닙니다. Linker는 시스템 화면의 보이지 않는 HTML(DOM) 구조를 직접 분석하고 제어하는 현장 맞춤형 파이프라인입니다.
- **다중 창 동시 제어**: 사용자는 10개의 시스템 창을 띄워두고, 확장 프로그램이 제공하는 폼(Form)에서 단 한 번만 템플릿을 완성하면 됩니다. "이 내용으로 복사" 기능과 일괄 전송 알고리즘이 10개의 창에 데이터를 수 초 내에 동시에 꽂아 넣습니다.
- **캘린더 피커 내장**: 텍스트로 치기 번거로운 제공일시(YYYY-MM-DD)를 OS 네이티브 캘린더 피커를 통해 직관적으로 선택하도록 변환했습니다.

### Value 2. 인지적 마찰을 없앤 클릭 제로 UI (Zero-Friction)
입력 과정 자체를 놀이처럼 수월하게 만들기 위해 사용자 경험(UX)의 미세한 마찰음까지 제거했습니다.
- **마우스 오버 자동 전개 (Hover-to-Open)**: 드롭다운 메뉴를 클릭해서 열 필요가 없습니다. 마우스를 올리는 즉시 메뉴 항목이 나열되며, 즉각적인 선택이 가능합니다. 이로 인해 마우스 피로도가 60% 이상 절감됩니다.
- **리스트 빌더 (List Builder)**: 기존의 불안정한 체크박스나 다중 클릭 방식 대신, 선택 즉시 화면 하단에 칩(Chip) 형태로 명단이 쌓이는 가시성 높은 컴포넌트를 자체 개발하여 인지적 안정성을 제공합니다.

### Value 3. 휴먼 에러 원천 차단 (Zero-Defect)
사회보장정보시스템과 같은 국가 중요 데이터베이스에서 오입력은 단순한 실수가 아니라 법적/행정적 문제로 이어질 수 있습니다.
- **비동기 순차 기입 시스템 (Sequential Processing)**: 담당자나 대상자가 여러 명(다중 선택 요소)일 경우, 시스템이 인식하기도 전에 기입이 넘어가는 현상을 막기 위해 150밀리초 단위의 비동기 안전 지연(Delay)을 로직에 내재화했습니다. 누락율 0%를 지향합니다.
- **시각적 입력 검증 (Visual Feedback)**: 데이터가 성공적으로 꽂힌 필드는 '보라색 네온 테두리'로, 형식 오류나 입력 누락이 발생한 필드는 '빨간색 점선 테두리'로 직관적인 피드백을 주어 즉각적인 2차 교정을 유도합니다.

## 4. 제안 결론 (Conclusion)
**"우리 기관은 더 이상 시스템의 드롭다운을 스크롤하는 데 세금을 쓰지 않습니다."**

Linker는 단순히 업무를 약간 편하게 해주는 툴이 아닙니다. **월 5만 원, 연 60만 원**의 압도적으로 저렴한 예산으로 **기관 소속 모든 상담원들의 일평균 500분(시간외근무수당 환산 시 약 월 150만 원 이상의 가치)을 되찾아주는 투자**입니다. 
경직된 국가 시스템과 유연한 현장의 실무 사이를 이어주는 연결고리(Linker), 그것이 우리가 세상에 내놓을 제품의 진짜 이름입니다.
