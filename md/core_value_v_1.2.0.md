# 프로젝트 Dash 핵심 가치 제안서 (Core Value Proposition)
**버전**: v1.2.0 (엔드투엔드 파이프라인 통합 반영)
**작성일**: 2026-03-13
**문서 목적**: 전국 아동보호전문기관 상담원을 위한 업무 혁신 솔루션 'Dash'의 최종 진화 모델 정의 및 3대 핵심 가치 제안

---

## 1. Executive Summary: "현장에서 시스템까지, 멈춤 없는 데이터 흐름"
사회복지 업무의 본질은 기록에 있습니다. 하지만 현재 상담원들은 현장에서의 메모, 이동 중의 정리, 사무실에서의 상급자 컨펌, 그리고 마지막 DB 입력까지 **동일한 내용을 4~5번씩 반복해서 다루는 '데이터 파편화'**의 늪에 빠져 있습니다.

프로젝트 **Dash**는 단순한 '입력 보조기'를 넘어, 상담의 시작부터 DB 저장까지를 단일 파이프라인으로 연결하는 **'워크플로우 허브'**로 진화했습니다. AI 자동 정제와 실시간 연동 기술을 통해 전 과정의 비효율을 걷어내고, 상담원이 오직 '상담'에만 집중할 수 있는 환경을 구축합니다.

---

## 2. 워크플로우 변화상 (AS-IS vs TO-BE)

| 단계 | AS-IS (파편화된 프로세스) | TO-BE (Dash 통합 파이프라인) |
| :--- | :--- | :--- |
| **1. 현장 기록** | 클로바노트 녹음 혹은 수기 메모 | **Dash APP**에서 DB 실시간 셋팅 및 상담 기록 |
| **2. 데이터 정제** | 이동 중 메모장 등에 직접 요약 및 재정리 | Dash 내장 **AI 자동 정제** (상담 메모 -> DB용 소견) |
| **3. 협업/컨펌** | 정제된 내용을 복사해 메신저로 상급자 전송 | Dash **원클릭 공유** 기능을 통한 메신저 전송 |
| **4. DB 기입** | 컨펌된 최종본을 복사해 느린 시스템에 직접 타이핑 | PC 복귀 후 **Dash Extension**에서 리스트 선택 후 '주입' |
| **사용 도구** | 4~5개 (클로바노트, 메모장, 메신저, DB시스템 등) | **단 1개 (Dash)** |

## 3. 3대 기능적 핵심 가치 (Core Functional Values)

### Value 1. 엔드투엔드 통합 파이프라인 (Unified Documentation Pipeline)
Dash는 데이터가 생성되는 '현장'과 저장되는 '시스템' 사이의 모든 인위적인 단절을 제거합니다.
- **워크플로우 허브**: 현장에서 입력한 필드값이 사무실 PC의 확장 프로그램으로 실시간 동기화되어, 반복적인 데이터 재구성이 필요 없습니다.
- **업무 전진 배치**: 사무실로 돌아와서 시작하던 '정리 업무'를 현장과 이동 중에 마칠 수 있게 하여 야근을 원천적으로 차단합니다.

### Value 2. AI 기반 지능형 기록 정제 (AI-Powered Content Refinement)
단순한 텍스트 보관을 넘어, 시스템 기입에 최적화된 형태로 데이터를 가공합니다.
- **자동 포맷팅**: 구어체로 작성된 상담 메모를 AI가 분석하여, 사례관리 시스템의 소견란 양식에 맞는 정중하고 명확한 문체로 자동 변환합니다.
- **누락 방지**: 설정된 DB 필드(장소, 소요시간 등)가 누락되지 않도록 AI가 체크하여 기입 성공률을 높입니다.

### Value 3. 시스템 리스크 헷징 및 심리적 안전 (Anti-Fragile System)
불안정한 국가 시스템의 영향을 최소화하여 사용자의 스트레스를 해방합니다.
- **데이터 세이프 존**: 시스템 로그인 여부와 관계없이 Dash 앱에서 모든 작업을 완료하므로, 시스템 튕김에 의한 데이터 유실 위험이 0%입니다.
- **초고속 주입(Magazine Injection)**: 시스템 창을 열고 주입 버튼을 누르는 순간(최대 1초) 모든 작업이 완료되므로, 불안정한 시스템에 머무는 시간을 90% 이상 단축합니다.

---

## 4. 제안 결론 (Conclusion)
**"Dash는 상담원의 전문성을 '타이핑'이 아닌 '상담'에서 증명하게 합니다."**

Dash v1.2.0은 클릭 몇 번 줄이는 도구가 아닙니다. 사회복지 현장의 **'정보 생산 및 승인 프로세스' 전체를 재설계**하는 솔루션입니다. 파편화된 도구들을 Dash라는 하나의 파이프라인으로 통합함으로써, 연간 수백 시간의 행정 낭비를 줄이고 사회복지 서비스의 질을 한 차원 높이는 혁신을 제공할 것입니다.
