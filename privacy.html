대시(Dash) 개인정보 처리방침

시행일: 2026년 3월 1일

버전: 1.0

대시(Dash)는 크롬·엣지 브라우저 확장 프로그램 형태로만 제공되는 완전 로컬 클라이언트 사이드 서비스입니다.
회사는 어떠한 서버·데이터베이스·클라우드도 운영하지 않으며, 모든 데이터 처리와 저장이 이용자 본인의 브라우저 기기 내에서만 이루어집니다.

회사는 「개인정보 보호법」 제2조, 제15조, 제22조, 제29조 등 관련 법령을 준수하며, 개인정보를 최소화하고, 아동학대 관련 민감정보는 일체 수집·저장·처리·전송하지 않습니다.

1. 총칙

1.1 본 방침은 대시 확장 프로그램(Chrome Web Store 및 Edge Add-ons) 이용 시 적용됩니다.
1.2 대시는 서버가 없는 완전 로컬 서비스로, 이용자의 브라우저 메모리와 로컬 스토리지(localStorage) 안에서만 작동합니다.
1.3 국가아동학대정보시스템 내 모든 업무 데이터(아동·가족·행위자 정보, 제공내용 등)는 절대 수집·저장·전송되지 않습니다.
1.4 회사는 개인정보를 수집하지 않기 때문에, 별도의 회원가입·로그인·계정 관리가 없습니다.

2. 수집하는 개인정보의 항목

대시는 개인정보를 수집하지 않습니다.

구분 수집 여부 내용
회원가입 정보 (이메일, 이름 등) 수집 안함 회원가입 기능 자체가 없음
인증 정보 수집 안함 별도 인증 과정 없음
아동학대 관련 업무 데이터 수집 안함 엑셀 파일 내용, 입력값, 팝업 데이터 등 전부 로컬 메모리에서만 일시 처리 후 즉시 삭제
사용 로그 수집 안함 서버로 전송되는 로그 없음 (브라우저 콘솔에만 디버깅용 익명 로그 생성 가능)
브라우저 정보 최소 수집 확장 프로그램 정상 작동을 위한 브라우저 버전·OS 정보만 Chrome/Edge API를 통해 로컬에서 확인 (서버 전송 없음)

중요: 이용자가 입력하는 모든 업무 데이터(엑셀 파일, 드롭다운 선택값, 제공내용 등)는 브라우저 메모리 안에서만 일시적으로 존재하며, 작업 완료와 동시에 자동 삭제됩니다.

3. 개인정보의 수집·이용 목적

  • 확장 프로그램의 정상적인 작동 및 기능 제공
  • 이용자 기기 내에서만 이루어지는 DB 입력 자동화 지원

4. 개인정보의 보유 및 이용 기간

대시는 개인정보를 수집하지 않으므로 보유 기간이 없습니다.
모든 데이터는 작업 종료 즉시 브라우저 메모리에서 삭제되며, 브라우저 종료 시 완전히 소멸됩니다.

5. 개인정보의 파기

  • 수집하는 개인정보가 없으므로 파기 절차 자체가 없습니다.
  • 업무 데이터는 작업 완료와 동시에 자동 파기됩니다.

6. 개인정보의 제3자 제공

대시는 개인정보를 수집하지 않으므로 제3자에게 제공할 개인정보가 없습니다.

7. 개인정보 처리의 위탁

개인정보를 처리하지 않으므로 위탁하지 않습니다.

8. 이용자의 권리와 행사 방법

대시는 개인정보를 수집하지 않으므로 별도의 열람·정정·삭제 요청이 필요 없습니다.
다만 확장 프로그램을 완전히 제거하면 모든 로컬 데이터(설정값 포함)가 삭제됩니다.

9. 개인정보의 안전성 확보 조치

대시는 다음과 같은 조치를 통해 안전성을 확보합니다.

9.1 기술적 조치

  • 모든 처리 로컬 클라이언트 사이드에서만 수행
  • 민감 업무 데이터는 브라우저 메모리에만 일시 저장 후 즉시 삭제
  • 로컬 스토리지에 저장되는 설정값은 암호화되지 않은 단순 키·값 형태 (민감정보 없음)

9.2 관리적 조치

  • 개발자 본인만 코드 접근 가능
  • 정기적 코드 감사
